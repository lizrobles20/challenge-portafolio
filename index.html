Proyectos

Encriptador de texto

Proyecto hecho con HTML5, CSS3 y Javascript. Encripta cualquier texto reemplazando las vocales por alguna palabra corta. No se permite ingresar mayúsculas, acentos o cualquier caracter especial. También es posible desencriptar textos que ya han sido encriptados. Fue el primer challenge del programa Oracle Next Education con Alura Latam.

Repostería Catra

Proyecto hecho con HTML5, CSS3, Javascript, PHP y MySQL. Tienda en línea sobre repostería. Funciona creando una cuenta para ingresar a la tienda. Cuenta con un carrito funcional haciendo uso de LocalStorage, teniendo la opción de eliminar elementos. Se almacenan los detalles del pedido en una base de datos. Hay una pantalla de administrador el cual puede ver todos los pedidos guardados, visualizando los detalles ingresados por el cliente con anterioridad.

Sistema de evaluación y diagnóstico de padecimientos

Proyecto hecho con Java y MySQL con Netbeans IDE. La aplicación requiere inicio de sesión para ingresar al sistema. En este programa se pueden crear perfiles clínicos ingresando datos del paciente. Cada perfil se almacena en una base de datos con un ID. En la pantalla principal se despliegan los perfiles en una tabla, se puede accesar a cada perfil, visualizar los datos, actualizarlos o eliminarlos. Se puede generar un reporte en PDF con los datos de diagnóstico y tratamiento.
